    T1071.001  Web Protocols (C2)            – HTTP/S used for command-and-control

    T1486  Data Encrypted for Impact         – both use destructive payloads

    The shared use of T1486, T1078, and T1071 makes automated attribution

    infrastructure / ICS; Lazarus: financial institutions / cryptocurrency) and
    in secondary tools (Sandworm: Industroyer/Sandworm wiper; Lazarus: ELECTRICFISH,
    FASTCash). Shared TTPs do not imply shared origin – they reflect shared
    knowledge of what works against modern defences.

--- EXERCISE 2: Disinformation Analysis ---
Sample Case Analysed: Operation Secondary Infektion
  Primary source: Stanford Internet Observatory / EU DisinfoLab joint report (2020)
  https://secondaryinfektion.org

Indicators Found (5-indicator red-flag table):
  1. Coordinated inauthentic accounts posting identical content within minutes
  2. Newly registered domains mimicking legitimate news outlets (typosquatting)
  3. Forged documents attributed to real officials (detectable via metadata)
  4. Cross-platform amplification: content seeded on fringe forums, then cited by
     larger accounts as if independently discovered
  5. Linguistic patterns inconsistent with claimed author nationality
     (machine-translated syntax errors, unusual idiom choices)

OSINT Tool Used: WHOIS / DNS lookup via lookup.icann.org
  Query: one domain cited in the Secondary Infektion report
  Output: registration date post-dated the event being "reported", registrant
          privacy-shielded through a reseller in a jurisdiction with no WHOIS
          transparency requirements. This is consistent with infrastructure
          set up specifically for the operation rather than a long-standing outlet.
  Limitations: WHOIS data alone cannot prove intent; domain registration timing
               is a red flag, not proof. Full attribution requires corroborating
               network traffic, hosting pattern analysis, and content fingerprinting.

Assessment: Operation Secondary Infektion ran from at least 2014 to 2020, planted
  forged documents across 300+ platforms in 7 languages, and targeted audiences
  in Ukraine, Germany, France, Spain, and the UK. The desired cognitive effect was
  distrust of Western governments and NATO. The operation was eventually exposed
  because the same document templates, linguistic fingerprints, and infrastructure
  patterns appeared repeatedly – exactly the indicators in the table above.
